
# –ù–µ–ø—Ä–µ—Ä—ã–≤–Ω–æ—Å—Ç—å —Ä–∞–±–æ—Ç—ã –±–∏–∑–Ω–µ—Å–∞ (Business Continuity)
–ü—Ä–æ—Ü–µ—Å—Å—ã –∏ –ø—Ä–æ—Ü–µ–¥—É—Ä—ã –¥–ª—è –≥–∞—Ä–∞–Ω—Ç–∏–∏ –Ω–µ–ø—Ä–µ—Ä—ã–≤–Ω–æ—Å—Ç–∏ –±–∏–∑–Ω–µ—Å –æ–ø–µ—Ä–∞—Ü–∏–π. –†–µ—à–µ–Ω–∏—è –¥–ª—è **BC** –Ω–∞–ø—Ä–∞–≤–ª–µ–Ω—ã –Ω–∞ –ø—Ä–æ–±–ª–µ–º—ã –Ω–µ–¥–æ—Å—Ç—É–ø–Ω–æ—Å—Ç–∏ –∏ —Å–Ω–∏–∂–µ–Ω–Ω–æ–π –ø—Ä–æ–∏–∑–≤–æ–¥–∏—Ç–µ–ª—å–Ω–æ—Å—Ç–∏ –ø—Ä–∏–ª–æ–∂–µ–Ω–∏–π. ## –í–æ–∑–¥–µ–π—Å—Ç–≤–∏–µ –≤—Ä–µ–º–µ–Ω–∏ –ø—Ä–æ—Å—Ç–æ—è
- –°—Ä–µ–¥–Ω–µ–µ –≤—Ä–µ–º—è –º–µ–∂–¥—É —Å–±–æ—è–º–∏ (**MTBF**) = –æ–±—â–µ–µ –≤—Ä–µ–º—è —Ä–∞–±–æ—Ç—ã / —á–∏—Å–ª–æ —Å–±–æ–µ–≤
- –°—Ä–µ–¥–Ω–µ–µ –≤—Ä–µ–º—è –Ω–∞ –≤–æ—Å—Å—Ç–∞–Ω–æ–≤–ª–µ–Ω–∏–µ (**MTTR**) = –æ–±—â–µ–µ –≤—Ä–µ–º—è –ø—Ä–æ—Å—Ç–æ—è / —á–∏—Å–ª–æ —Å–±–æ–µ–≤
- –î–æ—Å—Ç—É–ø–Ω–æ—Å—Ç—å –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏–∏ (**IA**) = MTBF / (MTBF + MTTR) –∏–ª–∏ –≤—Ä–µ–º—è —Ä–∞–±–æ—Ç—ã / (–≤—Ä–µ–º—è —Ä–∞–±–æ—Ç—ã + –≤—Ä–µ–º—è –ø—Ä–æ—Å—Ç–æ—è)

- **RTO** (Recovery Time Objective) - —Ü–µ–ª–µ–≤–æ–µ –≤—Ä–µ–º—è –≤–æ—Å—Å—Ç–∞–Ω–æ–≤–ª–µ–Ω–∏—è, –ø—Ä–æ–¥–æ–ª–∂–∏—Ç–µ–ª—å–Ω–æ—Å—Ç—å –≤—Ä–µ–º–µ–Ω–∏ –ø—Ä–æ—Å—Ç–æ—è, –∫–æ—Ç–æ—Ä–æ–µ –±–∏–∑–Ω–µ—Å –º–æ–∂–µ—Ç –≤—ã–¥–µ—Ä–∂–∞—Ç—å.
